Project Description
Basic illustration and a jump start for my design career is what I would say about this project.This project basically puts in all the graphic design illustration done for the UMBC The Choice Program-Flying Fruit Kiosk.

  • Exploring graphic design techniques

Illustration where done with respect to products at the Flying Fruit Cafe.Print design starting from Napkin holder,breakfast order print menu,Print design for Airpot.This is what got my started with the graphic designing.
